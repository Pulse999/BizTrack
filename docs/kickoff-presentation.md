## Our Challenge

# The Business Problem

## Current Situation

- New users learn by trial and error
- Support team answers same questions repeatedly
- No way to verify user knowledge
- No formal training certification

## The Impact

❌ Users don't use features they paid for  
❌ More support tickets  
❌ Frustrated users  
❌ Slower onboarding  
❌ No proof of user competency  

**💡 Solution: BizTrack Learning Hub**

# Your Mission

## What You're Going to Build

**A Learning Management System (LMS) that:**

✅ Trains BizTrack users through video courses  
✅ Tests their knowledge with quizzes  
✅ Tracks their progress automatically  
✅ Issues certificates upon completion  
✅ Reduces support requests  

**This is REAL software for a REAL business need!**

---

# What You'll Build - Admin View

## Admin Features

### Course Management
- Create new training courses
- Add course descriptions
- Organize content

### Video Management
- Add training videos (YouTube/Vimeo)
- Order videos sequentially
- Update or remove videos

### Quiz Creation
- Create quizzes for each course
- Add multiple-choice questions
- Set passing scores (default 80%)

### Analytics Dashboard
- View user completion rates
- See average quiz scores
- Identify struggling users

---

# What You'll Build - User View

## User Features

### Course Catalog
- Browse available courses
- See course descriptions
- View their progress

### Video Learning
- Watch training videos
- Mark videos as complete
- Track progress through course
- Videos unlock sequentially

### Knowledge Testing
- Take quizzes after completing videos
- See immediate results
- Retake if score < 80%
- View correct answers

### Certificates
- Automatically generated on passing
- Download as PDF
- Includes BizTrack branding
- Shows completion date and score
